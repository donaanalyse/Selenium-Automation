Selenium Automation Framework

A Java-based Selenium WebDriver automation framework built with Maven and JUnit.

 Tech Stack
1. Java 1.8
2. Selenium WebDriver 4.43.0
3. JUnit 4.13.2Maven
4.ChromeDriver

 Test Cases
| Test | Description | Status |

| testGoogleSearch | Searches for Selenium WebDriver on Google and verifies results appear |  Pass |

 How to Run
1. Clone the repository
2. Open in Eclipse as Maven project
3. Right-click GoogleSearchTest.java
4. Run As → JUnit Test

 What the Test Does
1. Launches Google Chrome
2. Accepts cookie consent if present
3. Searches for Selenium WebDriver
4. Verifies search results are displayed
5. Closes the browser
